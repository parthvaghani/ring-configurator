# Diamond Models Directory

Place your diamond 3D models (GLB format) here.

## Required Files:
- `round.glb` - Round brilliant cut diamond
- `princess.glb` - Princess cut diamond
- `emerald.glb` - Emerald cut diamond
- `cushion.glb` - Cushion cut diamond
- `oval.glb` - Oval cut diamond

## Instructions:
See the main MODELS_GUIDE.md in the project root for detailed instructions on:
- Where to download free diamond models
- How to convert other formats to GLB
- Model requirements and specifications

## Current Status:
The app will automatically:
- Load GLB models if they exist in this directory
- Fall back to procedural geometry if models are missing
- Apply your custom diamond color to the loaded models

## Quick Start:
1. Download diamond models from Sketchfab or other sources
2. Convert to GLB format if needed
3. Rename to match the required file names above
4. Place in this directory
5. Refresh the browser

No code changes needed - models will load automatically!
